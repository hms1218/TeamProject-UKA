## 🚀 프로젝트 소개

`유기동물 센터`는 전국의 보호소, 유기동물, 입양/임보 정보 등 다양한 정보를 제공하는 플랫폼입니다.  
사용자는 **지도 기반 검색**, **실시간 제보**, **입양/임시보호 신청**, **게시판** 등의 기능을 편리하게 이용할 수 있습니다.
